# ✋ Hand Gesture Recognition System

A real-time hand gesture recognition system built using computer vision and machine learning techniques. This project enables touchless interaction by detecting and interpreting hand gestures through a webcam.

---

## 🚀 Features

* Real-time hand tracking
* Gesture detection & classification
* Touchless system control
* Lightweight and fast processing
* Easy to extend with new gestures

---

## 🛠️ Tech Stack

* Python
* OpenCV
* MediaPipe
* NumPy

---

## 📂 Project Structure

```
├── main.py
├── utils/
├── models/
├── requirements.txt
└── README.md
```

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/hand-gesture-recognition.git
cd hand-gesture-recognition
pip install -r requirements.txt
```

---

## ▶️ Usage

```bash
python main.py
```

Make sure your webcam is connected and working.

---

## 📸 How It Works

* Captures video using webcam
* Detects hand landmarks using MediaPipe
* Processes gestures using trained logic/model
* Outputs corresponding actions
